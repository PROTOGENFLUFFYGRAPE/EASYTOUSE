COMMANDS

Discord Wick Bot Guide

Reasons

Always include ?r reason when warning, banning, or timing out users.

Warns

W!warn (user id) ?r reason — Warn a user and log the case.

W!cases (case id) ?e ?r new reason — Edit a warning reason.

Bans

W!ban (user id) ?r reason — Permanently ban a user.

W!hackban (user id) — Ban user not in server.

W!unban (user id) ?r reason — Remove ban.

Kicks

W!kick (user id) — Remove a user from the server.

Timeouts

W!timeout (user id) ?t TIME ?r reason — Temporarily mute a user.

W!delttimeout (user id) — Remove a timeout.

User Info

W!i (user id) — Shows detailed user info.

W!cases — Most recent cases.

Warn a User

W!warn (user id) ?r reason — Issue a warning.

Timeout a User

W!timeout (user id) ?t TIME ?r reason

Remove a Timeout

W!delttimeout (user id)

Ban a User

W!ban (user id) ?r reason — ban a user.

W!hackban (user id) — Ban user even when they are not in server.

W!unban (user id) ?r reason — Removes the ban.

Mass Kick

W!kick USERID1 USERID2- mass kicks users

Mass Ban

W!Ban USERID1 USERID2 -mass bans the users
