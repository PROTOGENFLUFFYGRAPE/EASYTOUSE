COMMANDS

Discord Wick Bot Guide

User Info

W!i (user id) — Displays detailed information about a user.

User Cases

W!cases — Shows the most recent cases in the server.

W!cases (user id) — Shows all cases for that user.

Warn a User

W!warn (user id) ?r reason — Issues a warning and logs it.

Edit a Warning

Use W!cases or W!cases (user id) to find the case ID.

W!cases (case id) ?e ?r new reason — Edits the reason of a warning/case.

Timeout a User

W!timeout (user id) ?t TIME ?r reason — Temporarily restricts a user from chatting.

Remove a Timeout

W!delttimeout (user id) — Removes a timeout.

Ban a User

W!ban (user id) ?r reason — Permanently bans a user.

W!hackban (user id) — Bans a user even if not in the server.

Unban a User

W!unban (user id) ?r reason — Removes a ban.

Edit a Ban

Bans cannot be directly edited. Unban then re-ban with new reason:

W!unban (user id) ?r reason

W!ban (user id) ?r new reason

Mass Kick

W!kick USERID1 USERID2 — Kicks multiple users at once.

Mass Ban

W!ban USERID1 USERID2 — Bans multiple users at once.

Mass Timeout

W!timeout USERID1 USERID2 ?t TIME ?r reason — Timeouts multiple users at once.
