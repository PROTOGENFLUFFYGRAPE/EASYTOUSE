COMMANDS

Discord Wick Bot Guide

CHEET SHEET

Quick Overview

W!warn — Warns a user.

W!timeout — Temporarily mutes a user.

W!ban — Permanently bans a user.

W!kick — Removes a user from the server.

W!cases — View moderation logs.

Syntax Help

?r = Reason

?t = Time

(user id) = Target user's Discord ID

User Info

W!i (user id) — Displays detailed information about a specific user.

W!cases — Shows the most recent moderation cases.

W!cases (user id) — Shows all moderation cases related to that user.

Warn a User

W!warn (user id) ?r reason — Issues a warning to a user.

Edit a Warning

Use W!cases to find case ID.

W!cases (case id) ?e ?r new reason — Edits a warning.

Timeout a User

W!timeout (user id) ?t TIME ?r reason — Times the user out.

Remove a Timeout

W!delttimeout (user id) — Removes the timeout.

Ban a User

W!ban (user id) ?r reason — Permanently bans the user.

W!hackban (user id) — Bans user not in server.

Unban a User

W!unban (user id) ?r reason — Removes a ban.

Edit a Ban

You must unban then re-ban with new reason.

Mass Kick

W!kick USERID1 USERID2

Mass Ban

W!ban USERID1 USERID2

Mass Timeout

W!timeout USERID1 USERID2 ?t TIME ?r reason
