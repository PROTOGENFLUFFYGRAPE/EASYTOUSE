Wick Bot Commands

Discord Wick Bot Guide

User Information

W!i (user id) — Gives info about the user ID.

W!cases — Shows the most recent cases.

W!cases (user id) — Shows that user's cases.

Warn a User

W!warn (user id) ?r reason — Warns the user.

Edit Warns

1. Use W!cases to see recent cases.

OR

W!cases (user id) to see that user’s cases.


Edit using:

W!cases (case id) ?e ?r new reason

Timeout User

W!timeout (user id) ?t TIME ?r reason

Ban User

W!ban (user id) ?r reason — Bans the user.

W!hackban (user id) — Bans a user not in the server.

Unban User

W!unban (user id) ?r reason — Unbans the user.

Edit a Ban

There is no direct way to edit a ban.

You must:

W!unban (user id) ?r reason

Then:

W!ban (user id) ?r new reason

Mass Kick

W!kick USERID1 USERID2

Mass Ban

W!ban USERID1 USERID2

Mass Timeout

W!timeout USERID1 USERID2 ?t TIME ?r reason
